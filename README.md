# esp8266-ProMicro-Joystick
Remote control cursor with joystick
## Connection
### esp32 - Arduino JoyStick
- 3.3V -> VCC
- GND -> GND
- VRx -> D34
- VRy -> D35
- Button (SW) - D32

### D1 mini (esp8266) - Arduino Pro Micro
- 5V -> RAW
- GND -> GND
- D8 -> TX0
- D7 -> RX1

- Upload codes and u re ready to go :D
